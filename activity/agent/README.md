# LLM Agent

This activity provides an LLM Agent powered by cloudwego/eino, supporting tool-calling through a ReAct-based workflow.

## Installation

### Qodeflow CLI
```bash
qodeflow install github.com/postqode/qodeflow-component/activity/agent
```

## Configuration

### Settings:
| Name             | Type | Description
|:---              | :--- | :---
| provider         | string | Model provider (openrouter, postqode)
| model            | string | The model name (e.g. anthropic/claude-3-5-sonnet, gemini-2.0-flash)
| apiKey           | string | Provider API key
| systemPrompt     | string | Optional system-level instruction for the agent
| responseStructure| any   | Specifies the JSON schema structure the agent should respond with

### Input:
| Name    | Type   | Description
|:---     | :---   | :---
| message | string | User message to send to the agent

### Output:
| Name     | Type   | Description
|:---      | :---   | :---
| response | string | Text response generated by the agent

## Examples

### Basic Agent Chat
The below example sends a simple message to the agent:

```json
{
  "id": "agent_chat",
  "name": "Agent Chat",
  "activity": {
    "ref": "github.com/postqode/qodeflow-component/activity/agent",
    "settings": {
      "provider": "openrouter",
      "model": "anthropic/claude-3-5-sonnet",
      "apiKey": "your_api_key_here"
    },
    "input": {
      "message": "Hello, can you help me with a coding question?"
    }
  }
}
```

### Agent with System Prompt
The below example configures the agent with a system prompt:

```json
{
  "id": "agent_with_system_prompt",
  "name": "Agent with System Prompt",
  "activity": {
    "ref": "github.com/postqode/qodeflow-component/activity/agent",
    "settings": {
      "provider": "openrouter",
      "model": "anthropic/claude-3-5-sonnet",
      "apiKey": "your_api_key_here",
      "systemPrompt": "You are a helpful coding assistant. Always provide examples in Go."
    },
    "input": {
      "message": "How do I read a file?"
    }
  }
}
```

### Agent with Structured Response
The below example forces the agent to respond with a specific JSON structure:

```json
{
  "id": "agent_structured_response",
  "name": "Agent Structured Response",
  "activity": {
    "ref": "github.com/postqode/qodeflow-component/activity/agent",
    "settings": {
      "provider": "openrouter",
      "model": "anthropic/claude-3-5-sonnet",
      "apiKey": "your_api_key_here",
      "responseStructure": {
        "type": "object",
        "properties": {
          "answer": {
            "type": "string",
            "description": "The final answer"
          },
          "confidence": {
            "type": "number",
            "description": "Confidence score from 0-1"
          }
        },
        "required": ["answer", "confidence"]
      }
    },
    "input": {
      "message": "What is the capital of France?"
    }
  }
}
